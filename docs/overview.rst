.. _overview:

Architecture overview
=====================

These documents describe the library structure and the rationale behind its design. We start by presenting the
rationale behind the library design, then we present the data model and data flow, which define a grammar to
apply transformations to MS data. Finally, we describe the backend utilities, which orchestrates workflow
executions following the data flow.
