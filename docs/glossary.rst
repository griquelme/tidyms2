Glossary
========

.. glossary::

    assay
        A collection of samples measured in experimental conditions.

    envelope
        A collection of isotopologues features.

    EIC
        Extracted Ion Chromatograms

    feature group
        A collection of features detected in different samples associated to the same chemical
        identity using a feature matching algorithm.

    LC-MS
        Liquid Chromatography coupled to Mass spectrometry.

    m/z
        Mass to charge ratio

    ROI
        Region of Interest. A region extracted from a raw data file which may contain features.
        
    SNR
        signal-to-noise ratio
